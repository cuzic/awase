# .yab 配列エディタ設計書

## 概要

.yab ファイルをキーボード配列のビジュアル表示で編集するツール。
awase-settings とは別ウィンドウ（別バイナリ）として提供する。

## 起動方法

- コマンドライン: `awase-yab-editor [ファイルパス]`
- awase-settings のタブ1「レイアウト」横の [編集] ボタンから起動
- トレイアイコン右クリック → 「配列エディタ」

## 画面要素

### ツールバー

| ボタン | 動作 | ショートカット |
|--------|------|--------------|
| 開く | ファイル選択ダイアログ → .yab 読み込み | Ctrl+O |
| 保存 | 現在のファイルに上書き保存 | Ctrl+S |
| 名前を付けて保存 | ファイル選択ダイアログ → 別名保存 | Ctrl+Shift+S |
| 再読み込み | ファイルを再読み込みして表示を更新 | F5 |
| キーボードモデル | JIS / US ドロップダウン | — |

### 面タブ

4つの面を切り替え。選択中の面がハイライト。

| タブ | .yab セクション |
|------|----------------|
| 通常面 | ローマ字シフト無し |
| 左親指シフト | ローマ字左親指シフト |
| 右親指シフト | ローマ字右親指シフト |
| 小指シフト | ローマ字小指シフト |

### キーボードグリッド

物理キー配列を模したグリッド表示。

- 各セルに割り当てられた値を表示（かな文字 or ローマ字 or 特殊キー記号）
- セルの色分け:
  - 白: ローマ字
  - 薄青: リテラル
  - 薄緑: 特殊キー（後/逃/入/空/消）
  - 灰色: なし（無）
- 選択中のセルは太枠でハイライト
- セルクリックで選択 → 下部の編集パネルに詳細表示

#### グリッドサイズ

| モデル | Row 0 | Row 1 | Row 2 | Row 3 |
|--------|-------|-------|-------|-------|
| JIS | 13 | 12 | 12 | 11 |
| US | 13 | 12 | 11 | 10 |

#### セル表示

| YabValue | 表示 |
|----------|------|
| Romaji { romaji: "ka", kana: Some('か') } | か |
| Romaji { romaji: "sya", kana: None } | sya |
| Literal("ー") | ー |
| Special(Backspace) | ⌫ |
| Special(Enter) | ⏎ |
| Special(Escape) | ESC |
| Special(Space) | ␣ |
| Special(Delete) | DEL |
| None | — |

### 編集パネル（下部）

選択中のセルの詳細を編集。

| 項目 | UI | 説明 |
|------|---|------|
| 位置 | 表示のみ | (row, col) と物理キー名 |
| 種別 | ラジオボタン | ローマ字 / リテラル / 特殊キー / なし |
| 値 | テキスト入力 | ローマ字: "ka"、リテラル: "ー"、特殊キー: ドロップダウン |
| かなプレビュー | 表示のみ | ローマ字→かな変換結果（kana_table で解決） |
| [適用] | ボタン | 編集内容をグリッドに反映 |

#### 種別ごとの入力 UI

**ローマ字モード:**
```
  値: [ka        ]
  かな: か
```
- テキスト入力。全角で入力すると自動で半角に変換
- かな変換テーブルでプレビュー表示

**リテラルモード:**
```
  値: [ー        ]
```
- テキスト入力。Unicode 文字をそのまま入力

**特殊キーモード:**
```
  値: [Backspace ▼]
```
- ドロップダウンから選択（Backspace/Escape/Enter/Space/Delete）

**なし:**
- 入力フィールドなし。割り当て解除。

| 表示 | 内容 |
|------|------|
| ファイル名 | "nicola.yab" |
| 変更状態 | "変更あり" / "保存済み" |
| キーボードモデル | "JIS (13-12-12-11)" |

## 操作フロー

### ファイルを開く

```
[開く] ボタン or Ctrl+O
  ↓
ファイル選択ダイアログ（.yab フィルター）
  ↓
YabLayout::parse(content, model) で読み込み
  ↓
4面のグリッドに反映
  ↓
タイトルバーにファイル名表示
```

### セルを編集する

```
グリッドのセルをクリック
  ↓
編集パネルに現在値を表示
  ↓
種別を選択 → 値を入力
  ↓
[適用] ボタン
  ↓
グリッドに反映（セルの色も更新）
  ↓
ステータスバー「変更あり」
```

### 保存する

```
[保存] ボタン or Ctrl+S
  ↓
YabLayout → .yab テキスト形式にシリアライズ
  ↓
ファイルに書き込み
  ↓
ステータスバー「保存済み」
  ↓
awase 本体に WM_RELOAD_CONFIG を PostMessage（任意）
```

### 再読み込み

```
[再読み込み] ボタン or F5
  ↓
未保存の変更がある場合:
  「変更が保存されていません。再読み込みしますか？」ダイアログ
  ↓
ファイルを再読み込み → グリッドに反映
```

## .yab シリアライズ

YabLayout → .yab テキスト形式の変換が必要（現在はパースのみ）。

```rust
impl YabLayout {
    /// YabLayout を .yab 形式のテキストに変換する
    pub fn serialize(&self, model: KeyboardModel) -> String {
        let mut out = String::new();
        out.push_str(&format!("{}\n", self.name));

        serialize_face(&mut out, "ローマ字シフト無し", &self.normal, model);
        serialize_face(&mut out, "ローマ字左親指シフト", &self.left_thumb, model);
        serialize_face(&mut out, "ローマ字右親指シフト", &self.right_thumb, model);
        serialize_face(&mut out, "ローマ字小指シフト", &self.shift, model);

        out
    }
}

fn serialize_face(out: &mut String, section: &str, face: &YabFace, model: KeyboardModel) {
    out.push_str(&format!("[{section}]\n"));
    let row_sizes = model.row_sizes();
    for (row, &cols) in row_sizes.iter().enumerate() {
        let values: Vec<String> = (0..cols)
            .map(|col| {
                let pos = PhysicalPos::new(row as u8, col as u8);
                match face.get(&pos) {
                    Some(YabValue::Romaji { romaji, .. }) => romaji_to_fullwidth(romaji),
                    Some(YabValue::Literal(s)) => format!("'{s}'"),
                    Some(YabValue::Special(sk)) => special_key_name(*sk),
                    Some(YabValue::None) | None => "無".to_string(),
                }
            })
            .collect();
        out.push_str(&values.join(","));
        out.push('\n');
    }
}
```

### ローマ字→全角変換

シリアライズ時に半角ローマ字を全角に戻す必要がある（.yab 形式は全角）。

```rust
fn halfwidth_to_fullwidth(ch: char) -> char {
    if ch.is_ascii() {
        char::from_u32(u32::from(ch) + 0xFEE0).unwrap_or(ch)
    } else {
        ch
    }
}
```

これは既存の `fullwidth_to_halfwidth` の逆操作。
